# ATM Angular App

This is a multi-page ATM simulation built with Angular.

## Features
- 4-digit PIN login
- Checking and savings accounts
- Deposit and withdraw functionality
- Error handling (invalid input, insufficient funds)
- Transaction receipt screen
- Transaction history with timestamps
- Protected routes using Angular guards

## Tech Used
- Angular (Standalone Components)
- TypeScript
- HTML/CSS
- Bootstrap

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `ng serve`
4. Open `http://localhost:4200`

## What I Learned
- Component-based architecture in Angular
- Routing and route guards
- State management using Angular services
- Handling user input and validation
- Debugging and connecting multi-page flows
